# VoidOS

VoidOS is een volledig custom-built concept operating system, ontworpen in Figma, gegenereerd via Cursor en geanimeerd met GSAP.

## Features
- Cinematische animaties
- Custom laptop model (Void)
- Desktop UI + notificaties
- GSAP-driven effects
